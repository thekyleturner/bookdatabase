CS 425: Databases Final Project
May 1st, 2017
Group 11

Summary
The database backend for this project was setup as a MySQL database on a LAMP (Linux, Apache, MySQL, and PHP) server configuration. The database was administered through phpMyAdmin throughout the project. The GUI front-end was built in C# with Visual Studio as a windows application.

Interface
The interface contains 4 forms:
	Form 1 (accessible by admins)
	Form 2 (accessible by teachers)
	Form 4 (accessible by students)
	Form 3 (login form)

Visual Studio Setup
For this project, we used the following versions of Visual Studio and related software to avoid MySQL errors:
	Visual Studio Community 2015
	Connector/Net 6.9.8
	MySQL for Visual Studio 1.2.6
